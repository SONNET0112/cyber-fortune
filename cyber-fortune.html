📖 使用说明

  1. 在顶部输入你的 DeepSeek API Key（从 DeepSeek 开放平台 获取，以 sk- 开头）
  2. 填写 今天做了什么 和 心情（越详细签文越准）
  3. 点击 「求签」 按钮，等待 AI 生成运势
  4. 查看签文等级、幸运色、幸运数字、宜忌事项和箴言
  5. 点击 「保存到本地」 将今日运势存入浏览器本地存储
  6. 点击 「查看历史」 可回顾过往保存的运势
💡 提示：API Key 仅存储在浏览器内存中，刷新页面后需重新输入。签文由 DeepSeek API 实时生成，每次结果都不同。
